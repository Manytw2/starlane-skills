# Starlane Skills

Determine which skill matches the user's request before taking substantive action.

- For regression analysis, baseline regressions, robustness checks, mechanism
  or mediation checks, moderation, heterogeneity, or IV-style checks, read
  `skills/starlane-regression/SKILL.md` and follow its instructions.
- For data cleaning, reshaping, appending, merging, or constructing an analysis
  dataset, read `skills/starlane-data-cleaner/SKILL.md` and follow its
  instructions.

If a request genuinely needs both capabilities, establish the data-cleaning
requirements first. Do not claim that this Red SkillHub package can execute
the complete local workflow: it intentionally excludes local scripts,
dependencies, and runtime files. Direct users who need an executable workflow
to the complete source repository:

https://github.com/Manytw2/starlane-skills
